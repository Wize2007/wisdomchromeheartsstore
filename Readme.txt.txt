## Website Description

Wisdom ChromeHeart Store is a responsive multi-page e-commerce website developed using HTML5, CSS3, and JavaScript. The website showcases premium fashion products inspired by luxury streetwear and provides users with an interactive online shopping experience.

The website contains the following pages:

* Home Page
* About Us
* Products
* Services
* Contact Us

Key features include:

* Responsive navigation menu
* Product gallery
* Featured products
* Contact form
* Shopping cart simulation
* Dark mode
* Newsletter subscription
* Responsive layout
* Hover effects and animations
* Back-to-top button
* Live date and time display

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts
* Font Awesome Icons

## Browser Requirements

The website works correctly on modern web browsers, including:

* Google Chrome
* Microsoft Edge
* Mozilla Firefox
* Opera
* Brave

A screen resolution of at least **1366 × 768** is recommended for the best viewing experience.
